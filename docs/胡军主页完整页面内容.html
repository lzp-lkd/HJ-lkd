# 个人简介

::: {.content-card}

胡军，男，1976年10月出生，博士，教授，博士生导师。主要从事岩土工程数值仿真、尾矿库自动化安全监测及预警系统方面的研究。先后主持辽宁省教育厅重点项目、辽宁省教育厅一般项目和企业委托项目30余项，总经费3000余万元。以第一作者或通讯作者在国际、国内期刊发表论文60余篇，其中SCI、EI收录20余篇。已指导和正在指导的硕士研究生40人；先后荣获“省高校系统优秀共产党员”、“辽宁省优秀教师”、“鞍山市十大优秀青年”、“鞍山市十大杰出青年岗位能手”、“鞍山市优秀青年教师”、“鞍山市园丁创业能手”、“鞍山市师德建设标兵”等荣誉称号。

鞍山矿产资源丰富，矿山露天开采边坡安全问题、尾矿库安全监测预警和排岩场稳定性研究一直是企业面临的难题，团队受企业委托，完成了《西果园尾矿库在线监测系统完善升级工程》、《风水沟尾矿库160米标高以下在线监测系统完善升级工程》、《大球尾矿库165米标高以下在线监测系统完善升级工程》和《弓长岭露天矿排岩场边坡安全检测及预警》等诸多大型工程自动化安全监测及预警系统的研发及安装工作，目前系统运行稳定，监测的数据可靠，受到合作企业的一致好评。团队自主研发了基于摄影测量技术的尾矿库干滩监测系统，基于LORA无线网络的数据采集和传输系统，基于B/S架构的尾矿安全监测系统和尾矿库在线监测智能综合监控预警一体化平台等多项核心技术。

弓长岭露天铁矿是矿业公司主力露天矿之一，随着露天矿深凹开采，边坡的稳定性持续下降，对生产作业、运输及周边设施造成巨大威胁，已成为开采的瓶颈问题。通过现场调查、理论分析、室内试验、数值模拟和现场监测（TM30和GNSS）等手段研究边坡滑移机理，为边坡的防治提供技术支撑。研究成果对保障矿区安全生产具有重要意义，可带来巨大的经济、社会效益，具有广泛的应用前景。

# 研究方向

::: {.content-card}

胡军教授主要围绕岩土工程、矿山安全监测、尾矿库自动化监测与智能预警、露天矿边坡稳定性、数值仿真、多源数据融合和工程安全评价等方向开展教学、科研与工程应用研究。

:::

## 主要研究方向

::: {.research-grid}

::: {.research-card}
### 岩土工程数值仿真

面向岩体裂隙扩展、岩体渐进劣化、边坡失稳机制、尾矿坝渗流与应力耦合等问题，开展理论分析、数值模拟、室内试验与工程验证研究。
:::

::: {.research-card}
### 尾矿库自动化安全监测

围绕尾矿库坝体位移、浸润线、水位、干滩长度、坝体稳定性等关键指标，研究自动化采集、实时传输、在线分析与安全预警技术。
:::

::: {.research-card}
### 露天矿边坡稳定性分析

面向深凹露天矿开采条件下边坡稳定性下降、潜在滑移和灾变演化等问题，开展现场调查、理论分析、室内试验、数值模拟和现场监测研究。
:::

::: {.research-card}
### 多源数据融合与智能预警

研究基于多源监测数据融合、机器学习和智能分析的矿山工程安全评价方法，服务于尾矿库、露天矿边坡和排岩场安全预警。
:::

::: {.research-card}
### 摄影测量与干滩在线监测

自主研发基于摄影测量技术的尾矿库干滩监测系统，实现干滩长度等关键安全指标的在线测量与工程应用。
:::

::: {.research-card}
### 安全监测系统与平台开发

研发基于 LORA 无线网络的数据采集和传输系统、基于 B/S 架构的尾矿安全监测系统，以及尾矿库在线监测智能综合监控预警一体化平台。
:::

::: {.research-card}
### 岩石力学与声发射特征

研究裂隙岩体、复合岩体、冻融岩石、尾矿水侵蚀砂岩等材料的力学性质、损伤演化和声发射特征。
:::

::: {.research-card}
### 矿山工程安全与成果转化

面向鞍山地区丰富矿产资源开发中的露天矿边坡、尾矿库、排岩场等安全难题，开展工程应用研究和成果转化。
:::

:::
